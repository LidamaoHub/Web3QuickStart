# web3jsdemo

这个项目的几个案例将分别介绍使用web3.js组件连接用户钱包,切换网络,读取链上内容,写入链上内容的简单流程和简单代码实现

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
